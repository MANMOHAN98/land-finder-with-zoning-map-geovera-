GEOVERA

GEOVERA is a web-based platform developed to simplify the process of buying and selling land. The platform provides a secure environment where buyers can search for properties and sellers can list and manage their land details easily.

The main goal of GEOVERA is to make land transactions more transparent, accessible, and user-friendly through a digital platform.

1.Features
1.1 Buyer

* Search land based on location and details
* View property information
* Save interested properties
* Contact sellers

1.2 Seller

* Register land details
* Upload property information and images
* Manage property listings
* Track buyer inquiries



2.Technologies Used

* React.js
* Firebase Authentication
* Firestore Database
* Google Maps / OpenStreetMap
* Tailwind CSS

3. How to Run the Project

```bash
git clone <repository-url>
cd geovera
npm install
npm run dev
```

4.Project Objective

The purpose of this project is to provide a simple and efficient solution for land buyers and sellers. By bringing property information into a centralized platform, GEOVERA helps users discover, manage, and connect with confidence.

GEOVERA – Locate. Verify. Own.
